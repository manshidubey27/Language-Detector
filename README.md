Language Detector - Data Science Project

This project is a Natural Language Processing (NLP) tool designed to identify the language of a given text. Whether it's a single word, a sentence, or a paragraph,
the model can accurately detect the language from a diverse dataset.

-> How it Works
The project follows a structured data science pipeline to ensure high accuracy and efficiency:

1.Importing Data: Utilizing a multi-language dataset containing thousands of samples across various global languages.

2. Feature Extraction: Converting raw text data into numerical vectors using techniques like CountVectorizer or TF-IDF, allowing the machine learning model to
"read" the patterns in different languages.

3. Fitting the Model: The processed data is fed into a classification algorithm (such as Multinomial Naive Bayes or Logistic Regression) to train the model on
language-specific linguistic patterns.

4. User Input: An interactive interface where users can input any text, and the model predicts the language in real-time.
